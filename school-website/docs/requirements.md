# School Website Requirements

## Project Goal

Develop a modern, mobile-first, responsive school website with an administration dashboard, student portal, digital applications, and future expansion capabilities.

---

# User Roles

## Visitor (Guest)

Can:
- Browse website
- View notices
- View gallery
- Download PDFs
- View calendar
- Contact school
- Apply for admission
- Apply for vacancies

Cannot:
- Edit anything
- Access dashboards

---

## Student

Can:
- Sign in
- View profile
- Submit digital applications
- Track application status
- Receive notifications
- Download documents

---

## Administration

Can:
- Sign in
- Publish notices
- Manage gallery
- Upload PDFs
- Manage calendar
- Manage applications

Cannot:
- Manage administrators
- Change system settings

---

## Super Admin

Can:
- Everything
- Manage admins
- Manage permissions
- Backup website
- View activity logs
- Change website settings

---

# Technical Requirements

- Mobile First
- Responsive Design
- Dark Mode
- Firebase Backend
- Secure Authentication
- Fast Loading
- SEO Friendly
